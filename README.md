## Install

- Via Go: `go get -u -v github.com/petermbenjamin/lolcat`
- Via [GitHub Releases](https://github.com/petermbenjamin/lolcat/releases)

## Usage

```
$ printf 'lolololololololol\n%.0s' {1..100} | lolcat
```
